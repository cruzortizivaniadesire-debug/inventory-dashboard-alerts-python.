 inventory-dashboard-alerts-python.
Automated Purchasing Dashboard &amp; Stock Alert System


 Why is this project critical for the business?
How much money does a business lose when purchasing teams have to read massive text reports to find out what is missing? Looking at spreadsheets raw line-by-line delays decisions, leading to late orders and costly out-of-stock situations. Buyers need data visualized instantly to capture urgent needs at a glance.

Even with unified data, the procurement team at GlobalMart wasted precious hours scanning row after row to identify which products were running low, often missing crucial restocking windows for top-selling items.

 I built an automated visual dashboard using **Python (Matplotlib & Pandas)**. The system reads the relational database and automatically generates an interactive color-coded health status report. It highlights critical inventory levels in **RED** (under 5 units) and healthy items in **GREEN**, alongside a real-time sales performance chart.

---

Tech Stack & Skills
Python (Matplotlib): Data visualization engineering and automated image reporting.
Python (Pandas): Data manipulation and logical classification conditions.
SQL: Relational query aggregation to power the visual metrics.
